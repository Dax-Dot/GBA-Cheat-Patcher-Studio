## Cheat database

Bundled CodeBreaker cheat data is attributed to GameHacking.org.

- GameHacking.org: https://gamehacking.org/
- MiSTer cheat source precedent: https://github.com/MiSTer-devel/Cheats_MiSTer
- MiSTer documentation notes that cheats are pulled from GameHacking.org:
  https://mister-devel.github.io/MkDocs_MiSTer/cores/features/cheats/

The bundled file `database/gba_codebreaker_named_db.json` was generated from a
GameHacking.org-derived source archive named `GBA_CodeBreaker_Cheats(2).zip`.
This project does not claim ownership of the underlying cheat codes.

## ROM metadata database

Bundled ROM metadata is attributed to No-Intro.

- No-Intro DAT-o-MATIC: https://datomatic.no-intro.org/

The bundled file `database/nointro_gba_db_export_crc.json` was generated from
`Nintendo - Game Boy Advance (DB Export) (20260531-074517).zip`.
It contains ROM identification metadata and hashes, not ROM data.
